# AJIO Clone - Full Stack E-Commerce

A full-stack e-commerce web application built to replicate the core features of AJIO.

## Tech Stack(tools used)

- Frontend:React.js (Vite), React Router, CSS
- Backend Node.js, Express.js
- Database MongoDB Atlas
- Storage Cloudinary
- Payment Stripe
- Email Resend API

## Features

- OTP based authentication
- Product management
- Cart and wishlist
- Checkout with COD and Stripe
- PDF invoice generation
- Admin panel with charts
- Order management with email notifications
